Module: UI

Quintus HTML5 Game Engine - UI Module The code in `quintus_ui.js` defines the `Quintus.UI` module, which adds in some easily accessible UI elements into Quintus. Depends on the `Quintus.Sprite` module. UI lets you create UI elements like containers, buttons and text elements.
